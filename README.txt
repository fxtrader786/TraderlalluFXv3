Golubhai Hosted Bot (Full Version)

1. Update 'config.py' with your bot token and Telegram user ID.
2. Deploy to Render.com using this structure.
3. Bot will send CALL/PUT signals every 2 minutes with 15-second early alerts.

Files:
- main.py (Main logic)
- config.py (Config variables)
- requirements.txt (Dependencies)
- render.yaml (Render deploy config)
